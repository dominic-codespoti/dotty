# Stress Test Data

This directory contains stress test scenarios.

## Test Files

### Stress Patterns
- `rapid-input.txt` - Rapid keystroke patterns
- `high-volume-ansi.txt` - High volume ANSI sequence test
- `rapid-resize.txt` - Window resize stress test
- `rapid-scroll.txt` - Scrolling stress test

### Resource Tests
- `memory-pressure.txt` - Memory allocation patterns
- `cpu-intensive.txt` - CPU intensive operations
- `gc-pressure.txt` - Garbage collection pressure test

## Usage

Used in performance and stress tests across the test suite.
